[1.14.2] 2022/02/15
=================================================================
*** NOTE: REQUIRES Ripple Software Version 1.14.x ***
=================================================================
Feature:          Trellis - Added ability to select whether Trellis 
                            disables global stim flag on a pin when 
                            stopping a pattern.

                            update automatically.

                            port used to open/write log files.

                            the default Save Map folder.

Bug Fix:          Trellis - *.ns3 files are no longer automatically
                            saved regardless of selection.
Bug Fix:          Trellis - Prevent Trellis from hanging and/or 
                            crashing when left (or right) clicking on 
                            NIP in Instrument Panel.
Bug Fix:          Trellis - Spike Scope now launches reliably.
Bug Fix:          Trellis - Trellis will now report if a needed network 
                            port required for the log file is able to 
                            open properly.
Bug Fix:          Trellis - Fixed error in min/max values when exporting 
                            EDF data from *.nfx files.
Bug Fix:          Trellis - Added Restore Default button to File Save UI.
Bug Fix:          Trellis - Set All Front Ends of same type option in 
                            Hardware Config screen will now only appear 
                            if a configurable property exists.
